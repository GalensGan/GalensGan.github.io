个人开源

  • SendMultipleEmail

    一个批量可批量发送不同内容邮件的工具。

  • developer-lazy-tool

    Do anything by one command

    一个将复杂的操作步骤聚合成一条命令语句的命令行工具，让装逼一直在路上。

  • MSAdminTest

    MSAddinTest 使开发者在不关闭 Microstation 的情况下，重新修改编译加载 DLL 库，在开发 Microstation 的 Addin 时，可以快速进行热重载和调试。

    你只需在 keyin 前加上 MSTest test 即可调用现有的 keyin 命令。

  • BenlteyPowerShortcuts

    使用户在 Bentley 产品中拥有和 CAD 一样的快捷键体验的插件。

  • ConfValidation

    以配置驱动的 .NET Core 数据验证器。
