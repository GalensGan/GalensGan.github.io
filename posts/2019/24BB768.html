2021/08/13 博客从 jekyll 引擎换成 hexo 引擎

用了这么久的 jekyll，为什么突然就换了呢？

其实换之前我也挣扎很久，因为我不舍得 github 对 jekyll 友好的支持，使用 jekyll，我增加或者修改完文章，直接推送就可以了，github 会自动进行编译。

今天微风正好，我想给自己的主题美化一下，既然要美化，当然要安装环境咯，开始下载 jekyll 的 windows 版本安装，安装完成后，运行不起来，google 无果后，又尝试用 docker 安装 jekyll 环境，安装完成后，还是跑不起来…

一怒之下，我决定换成 hexo 了，起码它环境好安装，插件多，NexT 主题也一直在维护。

初次使用 Hexo,又成功了趟了满地的坑，填啊填，终于填完了，于是才有了这段总结。

…
纵有千言万语，亦道不尽万般辛酸！

原引擎的主题放在当前仓库的 jekyll 分支中，有需要的可以自行食用。
