## git 基础

### 获取 Git 仓库

通常有两种获取 Git 项目仓库的方式：

1. **在已存在目录中初始化仓库**

   ``` 
   git init
   ```

2. **克隆现有的仓库**

   ``` 
   // 第一种方式
   git clone <url> 
   
   // 第二种方式,可以将目标目录进行重命名
   git clone <url> targetDirectoryName
   ```
